The Challenge

  • Understanding autonomous vehicle (AV) logic is complex.
  • Existing sims are either too abstract (2D) or too heavy.
  • Lack of accessible, web-based tools to visualize how cars "see."

Our Solution

A lightweight, 3D browser-based simulation.

It visualizes the "brain" of a self-driving car in real-time using raycasting sensors.

Key Features

Realistic 3D World

Procedural city, fog, sky, and dynamic lighting.

Smart Traffic

AI traffic that reacts to you and avoids collisions.

Interactive Physics

Drift, brake, and accelerate with realistic friction.

Immersive Audio

Synthesized engine & environment sounds (Web Audio API).

How It Works: The "AI"

1. LIDAR Sensors: Raycasting casts 5 rays to detect distance.

2. Steering Logic: Uses a feedback loop to steer away from the closest obstacle.

3. Speed Control: Proportional braking based on frontal danger distance.

Technology Stack

  • Engine: Three.js (WebGL)
  • Language: Vanilla JavaScript (Zero dependencies)
  • Audio: Web Audio API (No external assets)
  • Design: CSS3 Glassmorphism

Future Scope

  • Machine Learning (Reinforcement Learning) instead of rules.
  • Multiplayer driving.
  • Procedural infinite city generation.
